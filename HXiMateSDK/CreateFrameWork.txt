制作通用的话framework



参考网站： http://www.iloss.me/2011/11/28/基于xcode4的ios-framework-制作教程/
注意：  网站第五条c点中国 c.Drad Code Stripping  书写错误，应为c.Dead Code Stripping



编译通过并生成 模拟器版本和真机版本的 framework




使用命令 lipo -create 真机framework/库文件 模拟器framework/库文件 －output 库文件

解释： 将两个不同平台的库文件合并成一个库文件，大小为两者之和

使用命令 lipo －info 库文件 信息 are: armv7 i386


发布应用时 应该将库文件替换成真机平台下的framework。
